SmartArmStack (LGPLv3 packages)

This repository contains all LGPLv3 packages related to the SmartArm development.

The current version of the packages are compatible with ROS Noetic in Ubuntu 20.04 Focal.

Dependencies

  1. Ubuntu 20.04 Focal
  2. ROS Noetic
  3. DQ Robotics Development Branch (NOT RELEASE)
sudo add-apt-repository ppa:dqrobotics-dev/development
sudo apt-get update
sudo apt-get install libdqrobotics*
